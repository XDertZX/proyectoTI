# proyectoTI
microservicios tienda de instrumentos
Franco Fernandez
Carlos Miranda

Descripcion del proyecto

en este proyecto buscamos como equipo modelar un sistema de gestion de ventas de una tienda de instrumentos de musica y dejarlo funcionando para el momento de la entrega con una base de datos funcional (LARAGON) y corriendo sin problemas

Gestión completa de clientes:
Registrar clientes
Listar clientes
Buscar clientes por ID
Actualizar información de clientes
Eliminar clientes

Gestión completa de empleados:
Registrar empleados
Listar empleados
Buscar empleados por ID
Actualizar información de empleados
Eliminar empleados
Consultar ventas realizadas por un empleado

Gestión completa de productos:
Registrar productos
Listar productos
Buscar productos por ID
Actualizar productos
Eliminar productos
Filtrar productos por categoría
Filtrar productos por marca

Gestión de sucursales:
Listar sucursales
Buscar sucursales por ID
Actualizar información de sucursales
Eliminar sucursales
Implementación de API REST utilizando:
Spring Boot
Spring Web
Spring Data JPA
DTOs para transferencia de datos
Arquitectura en capas (Controller, Service, Repository)
Manejo de respuestas HTTP con ResponseEntity
Integración con base de datos relacional mediante JPA/Hibernate
