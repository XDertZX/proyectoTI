# proyectoTI
microservicios tienda de instrumentos
Franco Fernandez
Carlos Miranda

Descripcion del proyecto

en este proyecto buscamos como equipo modelar un sistema de gestion de ventas de una tienda de instrumentos de musica y dejarlo funcionando para el momento de la entrega con una base de datos funcional (LARAGON) y corriendo sin problemas

Funcionalidades 

-Generar reporte de venta conjunto a los detalles de esta 
-Generar Reporte de Empleados 
-Generar Reporte de CLiente 
-Generar Reporte de Productos 
-Modificar Empleados
-Modificar Clientes
-Modificar Productos
-Agregar Empleado
-Agregar Clientes
-Agregar Productos
-Eliminar Empleado
-Eliminar Cliente
-Eliminar Producto
-Eliminar Sucursal
-Consultar Ventas por Empleado 
-Consultar Sucursal por ID
-Consultar Empleado por ID 
-Consultar Cliente por ID
